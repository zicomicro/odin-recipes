# odin-recipes

This is a simple recipe website built as part of **The Odin Project – Foundations**.

## About the Project

The goal of this project is to practice basic HTML by building a multi-page website that contains recipes.

Each recipe page includes:

* A title
* An image
* A description
* A list of ingredients
* Step-by-step instructions

The homepage links to all recipe pages.

## Built With

* HTML5
* Basic Git & GitHub workflow

## Project Structure

```text
odin-recipes/
├── index.html
├── recipes/
│   ├── pizza.html
│   ├── baked-potato-soup.html
│   ├── salami-sandwich.html
├── images/
└── README.md
```

## How to Use

1. Open `index.html` in your browser.
2. Click on any recipe link to view the recipe page.
